API Endpoints
GET: Retrieve products with pagination./api/products/?page={page}/
POST: Create a new product (Basic Auth Requied)./api/products/
PATCH: Update a product (Basic Auth Requied). /api/products/{id}/
PUT: Replace a product (Basic Auth Requied). /api/products/{id}/
GET: Retrieve All Category./api/category/
GET: Retrieve Specific Category./api/category/{id}
GET: All Carts for Authenticated User./api/cart/
GET: Retrieve Cart for Authenticated User./api/cart/{user}/
PATCH: Add Cart Items./api/cart/{id}/{user}/?product={id}&operation=add
PATCH: Remove Cart Items./api/cart/{id}/{user}/?product={id}&operation=remove
GET: Retrieve Wishlist for Authenticated User./api/wishlist/
GET: Complete Wishlist for Authenticated User./api/wishlist/{user}/

PATCH: Remove Wishlist Items./api/wishlist/{id}/{user}/?product={id}&operation=remove

PATCH: Update Order Items./api/order/{id}/
